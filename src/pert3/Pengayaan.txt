Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:
1. extends adalah keyword di Java yang menunjukkan hubungan pewarisan/inheritance antara dua class. Artinya class anak (subclass) mewarisi semua sifat dan perilaku dari class induk (superclass).
   class Manajer extends Karyawan {
   fungsi:
   - Menunjukkan bahwa Manajer adalah turunan (subclass) dari Karyawan
   - Manajer mewarisi semua properti dan method dari Karyawan
   - Manajer otomatis punya variabel nama dan gajiPokok dari Karyawan
2. - Method Overloading: Membuat beberapa method dengan nama sama dalam satu class yang sama, tetapi dengan parameter yang berbeda (jumlah, tipe, atau urutan parameter).
   - Method Overriding: Mendefinisikan ulang method yang sudah ada di parent class di dalam child class, dengan nama dan parameter yang sama persis. Digunakan untuk mengubah perilaku method di class anak.
3. super adalah keyword yang merujuk ke parent class (superclass). Dalam constructor subclass, super() digunakan untuk memanggil constructor parent class, dan harus ditulis di baris pertama.
   Manajer(String nama, double gajiPokok, double tunjangan) {
       super(nama, gajiPokok);
       this.tunjangan = tunjangan;
   }
   fungsi:
   - Memanggil constructor parent class (Karyawan)
   - Mengisi nilai nama dan gajiPokok yang ada di parent class
   - Harus di baris pertama constructor child
4. - private: Variabel/method hanya bisa diakses dalam class itu sendiri. Child class tidak bisa mengakses variabel private parent.
   - protected: Variabel/method bisa diakses dalam class itu sendiri, package yang sama, dan child class (meskipun beda package).
   Jika variabel di Karyawan diubah jadi private, Manajer tidak bisa mengaksesnya langsung. Solusinya: buat getter method (method khusus untuk mengambil nilai) di parent class.
5. Inheritance adalah konsep OOP dimana class baru (child) bisa mewarisi sifat dan perilaku dari class yang sudah ada (parent).
    Keuntungan:
    - Code Reusability: tidak perlu menulis kode yang sama berulang kali
    - Maintainability: perubahan di parent class otomatis berlaku ke semua child class
    - Polymorphism: bisa memperlakukan objek child sebagai objek parent
